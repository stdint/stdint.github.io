今天初步体验了一下Ruby语言，获得了一点初级的认识。

- 动态类型语言
- 解释器语言
- 格式要求比Python宽松
- 纯面向对象的语言，所有的元素都是对象
- 类似C的语法结构

语法糖：
- 减少了很多花括号的使用
- 首字母大写代表常量，小写代表变量
- 自带两种打印方法puts和print，puts自带回车
- 简单的array unpacking，例子：
{% highlight ruby %}
first, second, third = ARGV
{% endhighlight %}
